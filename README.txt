Vendedor Toyo – Paquete para generar APK
----------------------------------------
Contenido:
- visitas.html (archivo principal)
- index.html (pantalla inicial)
- manifest.json
- Iconos Android

Instrucciones mañana:
1. Ve a un generador WebView a APK (ej: webintoapp.com, appsgeyser.com, etc.)
2. Sube este ZIP completo.
3. Selecciona que la app cargará un archivo LOCAL (no URL).
4. Elige index.html como archivo de inicio.
5. Genera tu APK.

Cualquier duda, pregúntale a ChatGPT :)
